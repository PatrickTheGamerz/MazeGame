How it works
On every page load, a small script calls the Wikipedia REST API’s page/random/summary endpoint, which returns a random article summary in JSON. The response is then rendered into the card on the left.
Tip
Want to keep exploring? Just refresh the page. Each reload triggers a new request and a new random topic.

This is a lightweight client-side demo using the public Wikipedia REST API. For more advanced usage, you can also use the MediaWiki Action API random module .
